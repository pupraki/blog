Mar 17, 2018
799字

　　「好哦，那麼愛穿，今晚讓你穿個夠。」

　　不消多久，主人攤開束腳袋讓我下半身穿進去，快速地拉緊上面四五個束帶；又將我兩隻手臂穿過上衣兩隻不開口的袖子，交叉到背後固定。最後再把上衣和褲袋用更多的束帶連接，一隻全身被綁緊緊，關在帆布囚牢裏的狗狗就大功告成啦。

　　又讓我再一次硬了。
